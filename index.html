Styles

H1: Main Heading

  • Font-weight: 700 (bold)
  • Font-size: 50px
  • Font-family: Arial, sans-serif

H2: Section Heading

  • Font-weight: 700 (bold)
  • Font-size: 36px
  • Font-family: Arial, sans-serif

H3: Subheading

  • Font-weight: 700 (bold)
  • Font-size: 20px
  • Font-family: Arial, sans-serif

P: Paragraph

  • Font-weight: 400 (regular)
  • Font-size: 14px
  • Font-family: Arial, sans-serif

Positioning

  • Margin Left: 50px for the page
  • Margin Right: 50px for the page
  • Margin: 10px for page elements
  • 1px solid borders
